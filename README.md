# Itera

Proyecto base en **Scala (sbt)** para tu trabajo final.

## Requisitos

- **JDK 17+** (recomendado)
- **sbt**

## Cómo ejecutar

```bash
sbt run
```

## Estructura

- `src/main/scala/IteraApp.scala`: punto de entrada del programa.

## Notas para Git / GitHub
